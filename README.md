### Bước 1: Clone dự án

### Bước 2: Chạy

> yarn install

### Bước 3: Copy file .env.example -> .env và thêm các biến cần thiết

ZALO_SECRET_KEY=

ZALO_APP_ID=

### Bước 4: Chạy lệnh `yarn dev` và post message vào link http://localhost:3000/zalo-webhook

-> Nếu trùng port 3000, vui lòng thay đổi khi cần thiết
